### Jahreszeiten
Pro Runde vergeht ein Monat
- Aufgeteilt in Jahreszeiten Frühling, Sommer, Herbst, Winter
- Pro Jahreszeit 3 Monate

### Wirtschaftsphase
- Nur im 1. Monat einer Jahreszeit
- Reihum, jeder Spieler ist nacheinander dran
#### Reihenfolge
- [[Ressourcen]] Sammeln
- Bauen / Ausbauen / Forschen / [[Einheiten#Rekrutierung]]
### Armeephase
- Jeden Monat
#### Version 1
- Alle schreiben [[Truppenbewegung]] auf
- Bewegungen werden gleichzeitig ausgeführt
- [[Schlacht]] werden geschlagen
#### Version 2
- Reihum jeder bewegt eine Truppe
- [[Schlacht]] werden geschlagen
